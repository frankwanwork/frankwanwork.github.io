    Common Mistakes:
    1. main() function type.
      2. Correct: 
      				int main()
				{
					......;

					return 0;
				}

				void main()
				{
					......;
				}
      Incorrect: 
      				main()
				{
					......;
				}
    2. Initialize a variable.

      3. int, float, double, these variables are required to have initialization before using it. More.

      Correct: 
      				double sum=0;
				int n=0;
				sum += 10;
				if(n != 0)
				{
					......;
				}
				else
				{
					......;
				}
      Incorrect: 
      				double sum;
				int n;
				sum += 10;
				if(n != 0)
				{
					......;
				}
				else
				{
					......;
				}
